## Description
Pizza Hunt is a progressive web application where users can post pizza recipes and participate in discussion. Pizzas are added through a form input with pizza options to choose from or the option to create your own. If the user loses internet connection and hits submit the input will be cached until internet access is available. Once the pizza is added, users are able to comment and discuss the recipe. Alternatively, users can reply to each other's comments. The cache storage function applies to the comment and reply comments as well. IndexedDB and the Cache API are used to enable this application to function offline as a PWA.

  ## Installation
  Node.js, NPM, and MongoDB must be installed on your computer. <br />
  <br />Clone the repo: <br />
      `git clone https://github.com/CFsylvester/budget-tracker.git` <br />
  <br />Navigate to the root directory and run: <br />
      `npm install` <br />
  <br />To start the server locally, run the following: <br />
      `npm start` <br />
  <br />Navigate to <b>localhost:3001</b> in your browser to open the application locally.
